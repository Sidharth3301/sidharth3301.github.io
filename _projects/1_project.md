DittoRenderer is a custom ray-tracing renderer developed as part of the Computer Graphics course at Saarland University. The renderer supports area lights, shading normals, alpha masking, thin-lens cameras, reflections, shadows, and global illumination.

We also created a custom scene render for the Rendering Competition organized by the [Computer Graphics group](https://graphics.cg.uni-saarland.de/courses/cg1-2023/index.html). Visit the project [page](https://bakhtiardev.github.io/mystic-hollow/) for more details.
